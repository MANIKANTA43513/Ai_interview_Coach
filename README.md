# AI Interview Coach 🚀

## What I Built
I built an AI Interview Coach chatbot that simulates a real interview experience.  
The chatbot asks interview-style questions and evaluates user responses by providing:

- Score (out of 10)
- Strengths
- Weaknesses
- Clear explanation of concepts
- Improved version of the answer

The goal was to create a focused and purpose-built chatbot rather than a generic chat interface.

---

## Why I Chose This Topic
I chose the interview coaching domain because many students struggle with interview preparation and lack real-time feedback.

Instead of building a general chatbot, I designed a system that mimics how a real interviewer behaves — asking questions, evaluating answers, and guiding improvements.

This makes the experience more practical and useful for users preparing for technical interviews.

---

## Product Thinking & UX
I focused on making the chatbot feel like a real product:

- Clean and simple chat interface
- Structured responses (score, strengths, weaknesses)
- Instant feedback after each answer
- Reset interview option for better usability
- Designed flow to simulate real interview experience

---

## Tech Stack
- Next.js
- JavaScript
- Tailwind CSS

---

## How It Works
- User starts interview by typing a message
- Chatbot asks questions or evaluates answers
- Based on keywords, it provides structured feedback
- Ensures a smooth conversational experience

## Note on AI Usage
This project focuses on product experience, interaction design, and structured feedback rather than relying entirely on external AI APIs.
